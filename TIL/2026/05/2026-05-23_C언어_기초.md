## 변수와 자료형 (정수/실수/문자열)

```c
#include <stdio.h>

int main(void) {
    int i = 42;
    double pi = 3.14159;
    char c = 'A';
    const char *s = "hello"; // 문자열 리터럴은 보통 포인터로 다룹니다.

    printf("i=%d, pi=%.2f, c=%c, s=%s\n", i, pi, c, s);
    return 0;
}
```

## 포인터 기본

- 포인터는 “메모리 주소”를 저장합니다.
- &x는 x의 주소, *p는 p가 가리키는 값을 의미합니다.
```c
#include <stdio.h>

int main(void) {
    int x = 10;
    int *p = &x;

    printf("x=%d\n", x);
    printf("p=%p\n", (void*)p);
    printf("*p=%d\n", *p);

    *p = 20; // x 값이 바뀜
    printf("x=%d\n", x);
    return 0;
}
```

## 배열과 문자열

```c
#include <stdio.h>

int main(void) {
    int a[3] = {1, 2, 3};
    printf("a[0]=%d, a[1]=%d, a[2]=%d\n", a[0], a[1], a[2]);

    char name[] = "Kim"; // 마지막에 '\0' 포함
    printf("name=%s, len(rough)=%zu\n", name, sizeof(name) - 1);

    return 0;
}
```

## 함수: 선언/정의/호출

```c
#include <stdio.h>

int add(int x, int y); // 선언

int add(int x, int y) { // 정의
    return x + y;
}

int main(void) {
    printf("add(2,3)=%d\n", add(2, 3));
    return 0;
}
```

## 구조체(struct) 간단 예제

```c
#include <stdio.h>

typedef struct {
    int id;
    const char *name;
} User;

int main(void) {
    User u = { .id = 1, .name = "HC" };
    printf("id=%d, name=%s\n", u.id, u.name);
    return 0;
}
```

## 빌드/실행(로컬)

```bash
# macOS/Linux (gcc/clang)
gcc -Wall -Wextra -O2 main.c -o app
./app

# Windows (MinGW 예시)
gcc -Wall -Wextra -O2 main.c -o app.exe
app.exe
```
