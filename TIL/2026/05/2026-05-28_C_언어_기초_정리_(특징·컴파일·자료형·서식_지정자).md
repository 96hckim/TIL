## 1) C 언어의 특징

- 유닉스(Unix) 시스템 개발을 위해 만들어진 언어
- 시스템 프로그래밍에 적합 (하드웨어/OS에 가까움)
- 이식성(Portability) 이 좋음 (플랫폼이 달라도 비교적 쉽게 옮길 수 있음)
- 함수 단위로 모듈화해 프로그램을 구성하기 좋음
---

## 2) 컴파일 과정 (3단계)

1. 전처리(Preprocess): 전처리 지시자에 따라 소스 파일 가공
1. 컴파일(Compile): 전처리된 파일을 기계어인 개체 파일(Object file)로 변환
1. 링크(Link): 개체 파일 + startup code 등을 결합하여 실행 파일 생성
> 흐름: 소스 파일 → 전처리된 소스 파일 → 개체 파일 + startup code → 실행 파일

---

## 4) 자료형 요약표 (크기·범위)  unsigned 포함

> 💡 크기/범위는 컴파일러·플랫폼에 따라 달라질 수 있습니다. 아래 표는 “학습용 대표값(많이 쓰는 환경 기준)”으로 정리했습니다.

| 분류 | 자료형 | 크기(바이트) | 저장 범위(부호 있음) | 저장 범위(unsigned) |
|---|---|---|---|---|
| 문자 | char | 1 | -128 ~ 127 | 0 ~ 255 (unsigned char) |
| 정수 | short | 2 | -32768 ~ 32767 | 0 ~ 65535 (unsigned short) |
| 정수 | int | 4 | -2147483648 ~ 2147483647 | 0 ~ 4294967295 (unsigned int) |
| 정수 | long | 4 (Windows) / 8 (Linux*64) | 환경 의존 | 환경 의존 |
| 실수 | float | 4 | 약 6~7자리 정밀도 | (부호 개념 없음) |
| 실수 | double | 8 | 약 15~16자리 정밀도 | (부호 개념 없음) |

## 5) printf/scanf 출력 변환 문자(서식 지정자) 표

| 구분 | 지정자 | 의미 | 예시 |
|---|---|---|---|
| 정수 | %d | 10진수(부호 있는 int) | printf("%d", i) |
| 정수 | %u | 10진수(unsigned int) | printf("%u", ui) |
| 정수 | %o | 8진수 | printf("%o", i) |
| 정수 | %x / %X | 16진수(소문자/대문자) | printf("%02X", i) |
| 문자 | %c | 문자 1개 | printf("%c", c) |
| 문자열 | %s | 문자열 | printf("%s", str) |
| 실수 | %f | 실수(float/double) 기본 출력 | printf("%.1f", d) |
| 실수 | %lf | scanf에서 double 입력 | scanf("%lf", &d) |
| 포인터 | %p | 주소 출력 | printf("%p", ptr) |
| 개행 | \n | Line Feed(다음 줄) | printf("Hi\n") |
| 탭 | \t | 탭 | printf("A\tB") |

### 6-1) Hello, World

```c
/*
	블럭 단위 주석(설명문)

	2. ver: 1.0
	3. 작성자: 김호철
	4. 작성일: 2026-05-28
	5. 기능: c언어 학습

#include <stdio.h> // printf, scanf, gets, fgets 등의 함수가 들어있다.
```

### 6-2) 자료형 크기/오버플로우/sizeof 예제

```c
#include <stdio.h>

int main(void)
{
	/*
		[용어 정리]
		- 변수(variable): 데이터(자료)를 저장하는 메모리(RAM) 공간
		- 상수(constant): 자료 값이 고정되어 변경할 수 없는 것 (RAM, ROM)

		[데이터 단위]
		- 니블 (Nibble): 4비트 (16진수 1자리를 표현)
		- 바이트 (Byte): 8비트 (영문자 1글자를 표현 가능한 기본 단위)
		- 워드 (Word): CPU가 한 번에 처리할 수 있는 데이터의 기본 단위
		  * 32비트 프로세서: 1워드 = 4바이트
		  * 64비트 프로세서: 1워드 = 8바이트

		[부호 비트]
		- MSB (Most Significant Bit): 최상위 비트
		  * 0: 양수, 1: 음수
	*/

	// 1) 문자형 (char)
	unsigned char uc = 0xff;        // 255
	unsigned char ub = 0b11111111;  // 255

	printf("hex uc ==> %u\n", uc);
	printf("bin ub ==> %u\n", ub);

	uc = 255;
	uc++; // overflow: 255 + 1 -> 0
	printf("uc++ ==> %u (0x%02X)\n", uc, uc);

	// 2) 정수형 (short)
	unsigned short us = 0xffff; // 65535
	printf("us ==> %u (0x%04X)\n", us, us);

	us++; // overflow: 65535 + 1 -> 0
	printf("us++ ==> %u (0x%04X)\n", us, us);

	// 3) sizeof
	int arr[4]; // 4byte * 4 = 16byte
	printf("\n--- sizeof ---\n");
	printf("char: %zu\n", sizeof(char));
	printf("short: %zu\n", sizeof(short));
	printf("int: %zu\n", sizeof(int));
	printf("long: %zu\n", sizeof(long));
	printf("double: %zu\n", sizeof(double));
	printf("arr[4]: %zu\n", sizeof(arr));

	return 0;
}
```

### 6-3) 배열 메모리 크기 예제

```c
#include <stdio.h>

int main(void)
{
	/*
		[메모리 정렬 팁]
		- 배열의 전체 크기(바이트)는 2의 배수(구체적으로는 4 또는 8의 배수)로 맞추는 것이 좋습니다.
		- CPU가 메모리에서 데이터를 읽어올 때 보통 4바이트 또는 8바이트 단위로 읽어오므로,
		  정렬이 맞으면 처리 속도가 좋아집니다.
	*/

	char c[6] = "abcd";             // 6
	unsigned char uc[20] = "abcde"; // 20

	short s[40];                    // 80
	unsigned short us[30];          // 60

	int i[10];                      // 40
	long l[20];                     // 80 (환경 의존)

	printf("c ==> %zu\n", sizeof(c));
	printf("uc ==> %zu\n", sizeof(uc));
	printf("s ==> %zu\n", sizeof(s));
	printf("us ==> %zu\n", sizeof(us));
	printf("i ==> %zu\n", sizeof(i));
	printf("l ==> %zu\n", sizeof(l));

	return 0;
}
```

### 6-5) ASCII 코드 테이블 출력

```c
/*
* ----|----|----|----|----|----|----|----|----|
*				ASCII CODE TABLE
*				================
* DEC		HEX			OCT			CHAR
* ===		===			===			====
* 48		0x30		060			'0'
*/
#include <stdio.h>

int main(void)
{
	const int total_chars = 128;
	const int lines_per_header = 30;

	for (int i = 0; i < total_chars; i++)
	{
		if (i % lines_per_header == 0)
		{
			printf("\n%10s%10s%10s%10s\n", "DEC", "HEX", "OCT", "CHAR");
			printf("%10s%10s%10s%10s\n", "===", "===", "===", "====");
		}

		if (i < 32 || i == 127)
		{
			printf("%10d%10X%10o%10c\n", i, i, i, ' ');
		}
		else
		{
			printf("%10d%10X%10o%10c\n", i, i, i, i);
		}
	}

	return 0;
}
```

### 6-6) 무한 루프(문자 입력 → ASCII 출력, 종료: q)

```c
#include <stdio.h>

int main(void)
{
	const int buffer_size = 10;
	char buff[10];

	while (1)
	{
		printf("임의의 문자 입력 (종료: 'q'): ");
		fgets(buff, buffer_size, stdin);

		if (buff[0] == '\n') continue;

		if (buff[0] == 'q')
		{
			printf("프로그램을 종료합니다.\n");
			break;
		}

		printf("입력한 문자: %c | DEC: %d | HEX: 0x%02X | OCT: %o\n\n",
			   buff[0], buff[0], buff[0], buff[0]);
	}

	return 0;
}
```

### 6-7) 입력 반복 (나이/키, 종료: 0 0)

```c
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>

int main(void)
{
	int age = 0;
	double height = 0.0;

	while (1)
	{
		printf("나이와 키를 입력하세요 (종료: 0 0) : ");
		scanf("%d %lf", &age, &height);

		if (age == 0 && height == 0.0)
		{
			printf("프로그램을 종료합니다.\n");
			break;
		}

		printf("나이: %d세 | 키: %.1lfcm\n\n", age, height);
	}

	return 0;
}
```
