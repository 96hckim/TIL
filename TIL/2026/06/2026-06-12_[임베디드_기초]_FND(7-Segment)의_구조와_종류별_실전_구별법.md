# 📑 FND 구조 및 구별법 마스터

> **FND(7-Segment)**는 내부에 들어있는 8개의 LED를 제어해 숫자를 표현하는 부품입니다. 이때 **내부 LED들의 극성을 한곳으로 어떻게 묶었느냐(Common)**에 따라 두 종류로 나뉩니다.

### 1. 💡 컴몬 캐소드 vs 컴몬 애노드 핵심 비교

FND는 어떤 신호를 주어야 켜지는지 제어 방식이 정반대이므로, 내가 가진 부품의 종류를 정확히 아는 것이 중요합니다.

| 구분 | 🔵 컴몬 캐소드 (Common Cathode) | 🔴 컴몬 애노드 (Common Anode) |
|---|---|---|
| 줄임말 | CC형 | CA형 |
| 공통 분모 | 모든 LED의 마이너스(-) 극을 묶음 | 모든 LED의 플러스(+) 극을 묶음 |
| 공통 핀 연결 | 보드의 **GND (0V)**에 연결 | 보드의 **VCC (5V / 3.3V)**에 연결 |
| LED 켜는 법 | 제어 핀에 High (1 / 5V) 신호 주입 | 제어 핀에 Low (0 / 0V) 신호 주입 |
| 특징 | 1 = ON이므로 직관적 (실습용 추천) | 0 = ON으로 반대 작동 (산업용 선호) |

### 2. 🔍 실전 부품 식별 방법 (3가지 꿀팁)

### 방법 ①: 파트 넘버(모델명) 확인하기 (가장 빠름)

- 부품 옆면/뒷면에 적힌 모델명을 구글에 검색(예: FND507 datasheet)하는 것이 가장 정확합니다.
- 제조사 관례상 모델명 끝자리에 C나 K가 있으면 Cathode(캐소드), A나 B가 있으면 Anode(애노드)형인 경우가 많습니다.
### 방법 ②: 멀티미터 "다이오드 모드" 테스트 (가장 확실함)

- 멀티미터를 🔄 다이오드/부저 모드에 놓습니다.
- 흑색 바늘(Negative)을 공통 자릿수 핀(D1~D4)에 대고, 적색 바늘로 데이터 핀(A~G)을 찔러서 불이 들어오면? ➡️ 컴몬 캐소드
- 적색 바늘(Positive)을 공통 자릿수 핀에 대고, 흑색 바늘로 데이터 핀을 찔러서 불이 들어오면? ➡️ 컴몬 애노드
### 방법 ③: 브레드보드 전원 직접 매핑 (계측기 없을 때)

> ⚠️ 주의: LED 보호를 위해 와이어 중간에 반드시 330옴 ~ 1k옴 저항을 직렬로 연결하고 찌르세요!

- FND 공통 핀에 GND를 묶고, 데이터 핀에 5V를 찔렀을 때 켜지면? ➡️ 컴몬 캐소드
- FND 공통 핀에 5V를 묶고, 데이터 핀에 GND를 찔렀을 때 켜지면? ➡️ 컴몬 애노드
