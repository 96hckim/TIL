비트 연산은 메모리를 극도로 아껴야 하는 임베디드 시스템, 네트워크 프로토콜 처리, 하드웨어 제어, 그리고 알고리즘 문제(Bit Masking)에서 필수적으로 사용됩니다.

# 📌 C언어 비트 연산자 종류 요약

비트 연산자는 크게 비트 논리 연산자(4개)와 비트 이동(시프트) 연산자(2개)로 나뉩니다.

| 연산자 기호 | 연산자 명칭 | 기능 설명 | 규칙 요약 |
|---|---|---|---|
| & | 비트 AND | 대응되는 비트가 둘 다 1이면 1 | 10 & 12 == 8 |
| | | 비트 OR | 대응되는 비트 중 하나라도 1이면 1 | 10 | 12 == 14 |
| ^ | 비트 XOR | 대응되는 비트가 서로 다르면 1 | 10 ^ 12 == 6 |
| ~ | 비트 NOT (반전) | 비트를 반대로 뒤집음 (1은 0으로, 0은 1로) | ~10 == -11 |
| << | 왼쪽 시프트 | 비트들을 지정한 횟수만큼 왼쪽으로 이동 | 빈자리는 0으로 채움 |
| >> | 오른쪽 시프트 | 비트들을 지정한 횟수만큼 오른쪽으로 이동 | 부호 비트 또는 0으로 채움 |

## 1. 비트 논리 연산자 직관적으로 이해하기

개념을 이해하기 위해 int a = 10; (2진수: 1010) 와 int b = 12; (2진수: 1100) 가 있다고 가정해 보겠습니다. (보기 편하게 4비트로만 표현합니다.)

### ① AND 연산 (&) : 특정 비트를 0으로 지우고 싶을 때 (Masking)

- 두 비트가 모두 1인 자리에만 1이 남습니다.
Plaintext

# 

```plain text
  1 0 1 0  (a = 10)
& 1 1 0 0  (b = 12)
---------
  1 0 0 0  --> 10진수로 8
```

### ② OR 연산 (|) : 특정 비트를 1로 켜고 싶을 때 (Setting)

- 두 비트 중 하나라도 1이 있으면 1이 됩니다.
```plain text
  1 0 1 0  (a = 10)
| 1 1 0 0  (b = 12)
---------
  1 1 1 0  --> 10진수로 14
```

### ③ XOR 연산 (^) : 두 데이터의 다른 점을 찾거나 토글(On/Off)할 때

- 두 비트의 값이 서로 다르면 1, 같으면 0이 됩니다. (똑같은 값을 두 번 XOR 연산하면 원래대로 돌아오는 특성이 있어 간단한 암호화에 쓰입니다.)
```plain text
  1 0 1 0  (a = 10)
^ 1 1 0 0  (b = 12)
---------
  0 1 1 0  --> 10진수로 6
```

### ④ NOT 연산 (~) : 비트 반전 (주의: 부호 비트가 바뀜)

- 0과 1을 완전히 뒤집습니다. C언어에서 정수는 컴퓨터가 '2의 보수' 알고리즘을 쓰기 때문에 ~a의 결과는 (a + 1) 수식의 결과와 같아집니다.
- ~10을 하면 결과는 -11이 됩니다.
## 2. 비트 이동 연산자 (Shift Operator)

비트들을 통째로 왼쪽이나 오른쪽으로 밀어버리는 연산입니다. 이 연산은 곱셈과 나눗셈을 초고속으로 처리할 때 많이 씁니다.

### ① 왼쪽 시프트 (<<) : 2^n 배수 곱하기

- a << 2는 a의 비트들을 왼쪽으로 2칸 밀어냅니다.
- 왼쪽으로 밀려 나간 비트는 버려지고, 오른쪽에서 새로 생기는 빈칸은 무조건 0으로 채워집니다.
- 수학적 의미: 한 칸 밀 때마다 값이 2배씩 증가합니다. (a << 2는 a x 2^2, 즉 a x 4와 같습니다.)
```plain text
  0 0 1 0 1 0  (값: 10)
       << 2 칸 이동
  1 0 1 0 0 0  (값: 40)
```



### ② 오른쪽 시프트 (>>) : 2^n 배수 나누기

- a >> 2는 a의 비트들을 오른쪽으로 2칸 밀어냅니다.
- 오른쪽 끝자리에서 밀려난 비트들은 버려집니다.
- 주의점 (왼쪽 빈자리 채우기): * 변수가 양수(Unsigned)라면 빈자리가 0으로 채워집니다.
- 수학적 의미: 한 칸 밀 때마다 값이 $\frac{1}{2}$배씩 감소(소수점 버림 나눗셈)합니다. (a >> 1은 a/2와 같습니다.)
## 📋 실무 팁 및 노션 핵심 메모

1. 비트 연산자는 관계 연산자(==, !=, <, >)보다 우선순위가 낮습니다!
1. 비트 연산은 연산 속도가 최상위권입니다.
```c
int main(void)
{
	int a = 0b00000000000000000000000000001010; // 10
	int b = 0b00000000000000000000000000001100; // 12
	int c = 0x80000001;

	// 1. 기본 비트 연산
	printf("a & b  : %d\n", a & b);   // AND (결과: 8)
	printf("a ^ b  : %d\n", a ^ b);   // XOR (결과: 6)
	printf("a | b  : %d\n", a | b);   // OR  (결과: 14)
	printf("~a     : %d\n", ~a);      // NOT (결과: -11)
	printf("a << 1 : %d\n", a << 1);  // Left Shift  (결과: 20)
	printf("a >> 2 : %d\n", a >> 2);  // Right Shift (결과: 2)

	printf("\n--- 특정 비트 리셋(Reset) 검증 ---\n");

	// 2. 3번 비트 끄기 (Reset)
	// [방법 #1] 하드코딩 마스크 사용
	a &= 0xFFFFFFF7;
	printf("a변수 3번 bit reset (하드코딩) : %.2X (10진수: %d)\n", a, a);

	// [방법 #2] 시프트 활용 동적 마스크 사용
	// 예) LED OFF
	b &= ~(1 << 3);
	printf("b변수 3번 bit reset (시프트)   : %.2X (10진수: %d)\n", b, b);

	printf("\n--- 특정 비트 원복(Set) 검증 ---\n");

	// 3. 3번 비트 다시 켜기 (Set)
	// 예) LED ON
	a |= (1 << 3);
	printf("a변수 3번 bit set (원복)       : %.2X (10진수: %d)\n", a, a);

	b |= (1 << 3);
	printf("b변수 3번 bit set (원복)       : %.2X (10진수: %d)\n", b, b);

	// 31번 bit reset
	c &= ~(1 << 31);
	printf("31번 bit reset : %.8X\n", c);

	// 31번 bit set
	c |= (1 << 31);
	printf("31번 bit set : %X\n", c);

	return 0;
}
```

```c
/*
	keyboard 1~12까지 입력 받기(scanf 활용)
	12~2: 겨울
	3~5: 봄
	6~8: 여름
	9~11: 가을 로 출력한다.

{
	int month;

	while (1)
	{
		printf("월을 입력하세요 (1~12) (프로그램 종료: 999): ");

		if (month == 999)
		{
			printf("프로그램 종료\n");
			break;
		}

		{

		case 2:
			printf("겨울\n");
			break;
		case 3:
		case 4:
		case 5:

			break;

		case 10:
		case 11:
			printf("가을\n");

		}
	}

	return 0;
```

```c
// [스위치 키] 1이면 초고속 매크로(MACRO) 사용, 0이면 일반 함수(FUNCTION) 사용
#define USE_MACRO 0 

// ==========================================
// 1. 매크로 함수 정의 (Compile 시점에 코드로 확장 배치)
// ==========================================
#define MACRO_ADD(a, b) ((a) + (b))
#define MACRO_SUB(a, b) ((a) - (b))
#define MACRO_MUL(a, b) ((a) * (b))
#define MACRO_DIV(a, b) ((a) / (b))

// ==========================================
// 2. 일반 함수 정의 (Call 호출에 의해 메모리 이동 후 실행)
// ==========================================
int func_add(int a, int b) { return a + b; }
int func_sub(int a, int b) { return a - b; }
int func_mul(int a, int b) { return a * b; }
int func_div(int a, int b) { return a / b; }

/**
 * @brief [통합 입력 함수] 사용자로부터 입력을 받아 첫 글자를 반환하고, 정수 변환 값도 제공한다.
 * @param title 콘솔에 출력할 안내 문구 (예: "연산자", "num1")
 * @param int_output 정수로 변환된 값을 저장할 변수의 주소
 * @return char 입력받은 문자열의 첫 번째 문자 ('+', '-', 'e' 등). 오류 또는 종료 시 0 반환.
 */
char get_input(const char* title, int* int_output)
{
	char input_buffer[16];

	printf("%s (종료: exit): ", title);
	if (fgets(input_buffer, sizeof(input_buffer), stdin) == NULL)
	{
		return 0;
	}

	// "exit" 입력 시 즉시 종료 신호(0) 반환
	if (strncmp(input_buffer, "exit", 4) == 0)
	{
		return 0;
	}

	// 만약 숫자가 들어왔을 경우를 대비해 정수 변환 값도 채워둠
	if (int_output != NULL)
	{
		*int_output = atoi(input_buffer);
	}

	// 입력받은 문자열의 첫 번째 글자를 반환 ('+', '-', '1' 등)
	return input_buffer[0];
}

int main(void)
{
	char op = 0;
	int num1 = 0;
	int num2 = 0;

	printf("=== 사칙연산 프로그램 ===\n");

	while (1)
	{
		// [1] 연산자 입력 (정수 변환이 필요 없으므로 NULL 전달)
		op = get_input("계산(+, -, *, /)", NULL);
		if (!op)
		{
			printf("프로그램을 종료합니다.\n");
			break;
		}

		// [2] 피연산자 num1, num2 입력 (통합 함수를 그대로 재활용)
		if (!get_input("num1", &num1)) break;
		if (!get_input("num2", &num2)) break;

		// [3] 사칙연산 분기 처리 및 출력
		printf("결과: ");
		switch (op)
		{
		case '+':
#if USE_MACRO
			printf("%d + %d = %d [MACRO]\n", num1, num2, MACRO_ADD(num1, num2));
#else
			printf("%d + %d = %d [FUNCTION]\n", num1, num2, func_add(num1, num2));
#endif
			break;

		case '-':
#if USE_MACRO
			printf("%d - %d = %d [MACRO]\n", num1, num2, MACRO_SUB(num1, num2));
#else
			printf("%d - %d = %d [FUNCTION]\n", num1, num2, func_sub(num1, num2));
#endif
			break;

		case '*':
#if USE_MACRO
			printf("%d * %d = %d [MACRO]\n", num1, num2, MACRO_MUL(num1, num2));
#else
			printf("%d * %d = %d [FUNCTION]\n", num1, num2, func_mul(num1, num2));
#endif
			break;

		case '/':
#if USE_MACRO
			printf("%d / %d = %d [MACRO]\n", num1, num2, MACRO_DIV(num1, num2));
#else
			printf("%d / %d = %d [FUNCTION]\n", num1, num2, func_div(num1, num2));
#endif
			break;

		default:
			printf("잘못된 연산자 기호입니다.\n");
			break;
		}
		printf("\n");
	}

	return 0;
}
```
