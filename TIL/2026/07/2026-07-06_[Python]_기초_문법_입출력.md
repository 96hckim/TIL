# 🐍 파이썬 기초 문법 핵심 정리

## ⏳ [Chapter A] 성능 측정 및 반복문 비교

> 파이썬에서 대량의 데이터를 처리할 때 내장 함수와 반복문(for)의 속도 차이를 비교합니다.

### [A-1] 천만까지 합 구하기 (for문)

- for문을 사용하여 0부터 10,000,000까지 1씩 더해가며 합계를 구하고, datetime을 이용해 실행 시간을 측정합니다.
```python
import datetime

tot = 0
start = datetime.datetime.now()

for i in range(10000001):
    tot += i

end = datetime.datetime.now()
print("time =", (end - start), tot)
```

### [A-2] 천만까지 합 구하기 (sum 내장 함수)

- 파이썬 내부적으로 최적화된 내장 함수 sum()과 생성자 range()를 사용합니다. 일반 for문보다 훨씬 빠른 속도를 보입니다.
```python
import datetime

start = datetime.datetime.now()

tot = sum(range(10000001))

end = datetime.datetime.now()
print("time =", (end - start), tot)
```

```python
print("Hello")
```

### [1-2] 파이썬의 주석

### [1-3] 들여쓰기 (Indentation)

- 파이썬은 문법적으로 들여쓰기가 매우 엄격한 언어입니다. 이유 없는 들여쓰기는 IndentationError를 발생시킵니다.

### [1-4] print 함수의 옵션 (sep, end)

- sep: 출력할 데이터들 사이에 넣을 구분자를 지정합니다. (기본값: 공백)
- end: 출력을 마친 후 끝에 무엇을 붙일지 지정합니다. (기본값: 줄바꿈 \n) end=" "를 주면 다음 print문이 줄바꿈 없이 옆으로 이어집니다.
```python
print("Hi", "Python") # 기본 공백 구분

print("Good", "Python", sep="***") # 데이터 사이에 *** 삽입

print("Hello")
print() # 빈 줄 출력
print("Python")

print("Welcome", "Python", sep=" ::: ", end=" ") # 줄바꿈 대신 공백
print("World!") # 위 줄 뒤에 바로 이어서 출력됨
```

## 🔢 [Chapter 2] 데이터 타입 및 변수

### [1-5] 정수형(int) 상수

- 파이썬은 아주 큰 정수도 글자 수 제한 없이 다룰 수 있습니다.
- 0b(2진수), 0o(8진수), 0x(16진수) 표현법을 지원합니다.
- 비교 연산의 결과인 True는 숫자 1과 같은 값을 가집니다.
```python
print(12346578901234567890) # 무한대 크기 정수 지원
print(15, 0b1111, 0o17, 0xF) # 모두 10진수로 '15'를 뜻함

print(True == 1, 5 > 10) # 출력: True False (True는 계산 시 1로 취급)
```

### [1-6] 실수형(float) 상수

- 소수점이 포함된 숫자입니다. 컴퓨터가 지수를 표현할 때 쓰는 e 기법도 지원합니다.
- 1e3은 1 x 10^3 = 1000.0, 1e-3은 1 x 10^{-3} = 0.001을 의미합니다.
```python
print(3.14, -3.14)
print(1e-3, 1e3) # 지수 표현식 (출력은 항상 실수형)
```

### [1-7] 변수 (Variable)

- 데이터를 담는 상자입니다. 파이썬은 변수의 대소문자를 구분하며(abc1과 Abc1은 다른 변수), 변수에 언제든 새로운 값을 덮어쓸 수 있습니다.
```python
abc1 = 10
Abc1 = abc1 + 1 # 대소문자 구분함
```

### [1-8] 변수의 타입 동적 할당

- 파이썬은 변수를 선언할 때 타입을 미리 정하지 않는 '동적 타이핑' 언어입니다. 변수에 담기는 값에 따라 타입이 언제든 변합니다.
```python
a, b, c = 10, 3.14, 'hello' # 여러 변수 동시 선언
print( type(a), type(b), type(c) ) # <class 'int'> <class 'float'> <class 'str'>

a, b, c = b, c, 100 # 변수의 값을 교환하거나 새로운 타입 할당 가능
print( type(a), type(b), type(c) )
```

## 🧮 [Chapter 3] 연산 및 문자열 다루기

### [1-9] 기본 산술 연산자

- : 거듭제곱, //: 나눗셈의 몫, %: 나눗셈의 나머지
- 문자열(str)끼리 더하면 글자가 이어지고, 문자열에 숫자를 곱하면 그 횟수만큼 반복됩니다.
```python
a, b = 10, 3
print( a**2, (-3)**2 ) # 거듭제곱 (100, 9)
print( a+b, a-b, a*b, a/b ) # 사칙연산 (/ 연산은 무조건 실수 반환)
print( 10//3, 10%3, 2.5//2, 2.5%2 ) # 몫, 나머지 (실수 연산도 가능)
print( 10+20/2, (10+20)/2 ) # 연산자 우선순위 (괄호가 먼저)

print('Hello' + ' ' + 'Python') # 문자열 연결
print('Hello' * 2) # 문자열 반복
```

### [1-10] 자릿수 분리

- 몫(//)과 나머지(%) 연산자를 활용하여 4자리 정수의 각 자릿수를 하나씩 추출하는 알고리즘입니다.
```python
n = 1234
d1 = n // 1000          # 천의 자리 (1)
d2 = (n // 100) % 10    # 백의 자리 (2)
d3 = (n // 10) % 10     # 십의 자리 (3)
d4 = n % 10             # 일의 자리 (4)
print(d1, d2, d3, d4)
```

### [1-11] 문자열 상수와 이스케이프 문자

- \n(줄바꿈), \t(탭 간격), \'(따옴표 표현) 등 기능을 가진 특수 문자 표현법입니다.
- 문자열 앞에 r을 붙인 Raw String은 내부의 이스케이프 문자를 기능하지 않고 문자 그대로 출력합니다.
```python
print( "Hi\nPyth\ton" ) # 줄바꿈과 탭 적용

print( r"Hi\nPyth\ton" ) # r 기호: 이스케이프 기능 제거 (그대로 인쇄)
```

## 🔄 [Chapter 4] 형변환 (Type Conversion)

### [1-13] 타입명 확인

- type() 함수 자체의 타입은 <class 'type'>이며, 특정 데이터의 타입을 검사하거나 강제로 형변환할 때 타입명을 함수처럼 사용합니다.
```python
print(type(int)) # <class 'type'>
print(type(10))  # <class 'int'>
a, b = int(3.14), int("10") # 다른 타입을 정수형(int)으로 변경
print(a, type(a), b, type(b))
```

### [1-14] 데이터 타입 변환 함수

- str(): 문자열로 변환, int(): 정수로 변환(소수점 아래 버림), float(): 실수로 변환
- bool(): 값이 비어있거나 0, 0.0 이면 False, 무언가 존재하거나 숫자가 있으면 True 반환
```python
print(str(10), str(3.14), str('apple'))
print(int(4.89), int('13'), int(-10)) # 실수는 소수점 버림
print(float(5), float('3.14'), float(1.78))
print(bool(1.5), bool(0.0), bool(True)) # 0.0은 False, 나머지는 True
```

### [1-15] 문자열 정수 연산

- 따옴표로 감싸진 "10"과 "20"은 숫자처럼 보이지만 문자열입니다. 글자끼리 더하면 "1020"이 되므로, 산술 연산을 하려면 int()를 통해 먼저 정수로 바꾸어 주어야 합니다.
```python
a, b = "10", "20"

# r = a + b  # ⚠️ 이렇게 하면 글자가 이어져서 "1020"이 됩니다.
r = int(a) + int(b) # 숫자로 형변환 후 덧셈 연산
print(r) # 출력: 30
```

### [1-16] 제품 판매가격 계산 (특판 기간 조건)

- 문자열 형식의 가격과 세율을 형변환하여 공식(p + p * t / 100)에 대입한 후, 1,000원 미만 절사(버림) 조건을 처리합니다.
- 가격을 1000으로 나눈 몫(// 1000)에 다시 1000을 곱하면 백의 자리 이하가 깔끔하게 버려집니다.
```python
p, t = "123456", "12.5"

# 1. 판매 가격 공식 계산 후 정수화
total = int(p) + int(p) * float(t) / 100
total = int(total)

# 2. 1,000원 미만 절사 연산 (예: 138888 -> 138000)
total = total // 1000 * 1000

print(total) # 출력: 138000
```

### [1-17] 키보드로 입력 및 타입 변환 (input)

- input() 함수는 사용자가 키보드로 입력한 값을 받아옵니다. 중요한 점은 무엇을 입력하든 무조건 문자열(str)로 받아온다는 것입니다.
- 따라서 숫자로 계산하려면 반드시 int()나 float()로 감싸서 형변환을 해주어야 합니다.
```python
x = input()        # 입력받은 그대로 문자열(str) 저장
y = int(input())   # 입력받은 글자를 정수(int)로 변환
z = float(input()) # 입력받은 글자를 실수(float)로 변환

print(x, type(x))
print(y, type(y))
print(z, type(z))
```

### [1-18] 두 정수 입력받아 합 구하기

- input()의 특성을 활용하여 키보드로 정수 2개를 각각 한 줄씩 입력받아, 숫자로 변환한 뒤 합계를 출력하는 실습입니다.
```python
# 사용자에게 두 정수를 연속으로 입력받음
a = int(input())
b = int(input())

# 정수로 변환되었기 때문에 글자가 이어지지 않고 수학적 덧셈이 수행됨
print(a + b)
```

### [1-19] split()을 이용한 이름 3개 입력받기

- input().split()은 사용자가 한 줄에 공백(띄어쓰기나 스페이스)을 두고 입력한 문자열을 쪼개서 각각의 변수에 나누어 담아줍니다.
- 이를 다중 할당(Multiple Assignment)이라고 하며, 입력받은 값의 개수와 변수의 개수가 정확히 일치해야 합니다.
```python
# 한 줄에 공백을 기준으로 3개의 단어를 입력받음 (예: 홍길동 김철수 이영희)
a, b, c = input().split()
print(a, b, c)
```

### [1-20] 콤마(,)로 분리된 2개의 정수를 한 줄에 입력받기

- split(sep=",") 또는 split(",")와 같이 괄호 안에 특정 문자를 지정하면, 띄어쓰기 대신 그 문자를 기준으로 문자열을 쪼갭니다.
- 문자열로 쪼개져 들어온 변수들을 산술 연산하기 위해 각각 int()로 형변환해 줍니다.
```python
# 한 줄에 콤마를 기준으로 2개의 정수를 입력받음 (예: 10,20)
a, b = input().split(sep=",")

# 문자열을 숫자로 형변환
a, b = int(a), int(b)
print(a + b)
```

### [1-21] 정수 3개를 한 줄에 받아서 합계 구하기

- split() 함수를 활용하여 한 줄에 공백으로 구분된 정수 3개를 한 번에 입력받고, 이를 모두 정수로 변환하여 총합을 구하는 실습 설계입니다.
```python
# 1. 한 줄에 공백으로 구분된 3개의 값 입력받기 (예: 10 20 30)
a, b, c = input().split()

# 2. 각각 정수형(int)으로 형변환
a, b, c = int(a), int(b), int(c)

# 3. 합계 계산 후 출력
s = a + b + c
print(s)
```

### [1-22] format() 메서드를 이용한 기본 인쇄

- 문자열 내부에 중괄호 {}를 치고 뒤에 .format()을 붙여 값을 순서대로 끼워 넣는 방식입니다.
- 중괄호 안에 인덱스 번호({0}, {1})를 지정해 순서를 바꾸거나 같은 값을 여러 번 쓸 수 있고, 이름을 지정({a})할 수도 있습니다.
```python
s1 = "{}, {}".format(10, 20)                 # 1. 순서대로 매칭
s2 = "{0} + {0} = {2}".format(2, 3, 2 + 2)    # 2. 인덱스 번호 지정 (0번째인 2를 재사용)
s3 = "a = {a}, b = {b}".format(a=10, b=20)    # 3. 이름을 직접 지정

print(s1, s2, s3, sep="\n")
```

### [1-23] format() 메서드 형식 지정 기호

- 중괄호 내부 콜론(:) 뒤에 규칙을 적어 숫자 포맷, 정렬, 공백 채우기, 소수점 자리 제한 등을 수행할 수 있습니다.
- {} 자체를 문자열로 출력하고 싶다면 중괄호를 두 번 {{{}}} 겹쳐서 사용합니다.
| 기호 예시 | 의미 | 실제 결과 (format(100) 등 기준) |
|---|---|---|
| {:d}, {:x}, {:f} | 정수형(d), 16진수(x), 실수형(f) 형식 출력 | 100 0x64 100.000000 |
| {:5s} | 최소 5칸의 자리를 확보한 문자열(s) | |Hi   | |
| {:5.1f} | 5칸을 확보하고 소수점 아래는 1자리만 표시 | |  1.2| |
| {:05.1f} | 5칸 확보 후 남는 빈자리를 0으로 채움 | |001.2| |
| :<5, :^5, :>5 | 각각 왼쪽 정렬, 가운데 정렬, 오른쪽 정렬 | Hi   , _Hi__, ***Hi |
| {:,} | 3자리마다 콤마(,) 추가 (천 단위 구분 기호) | 1,234 |
| {:.2%} | 백분율(%)로 변환하고 소수점 2자리까지 표시 | 24.50% |

```python
print("{0:d} 0x{0:x} {0:f}".format(100))
print("|{0:5s}|{0:5}|{1:2}|".format("Hi", "Good"))
print("|{0:5.1f}|{0:05.1f}|".format(1.23))
print("|{0:<5}|{0:_^5}|{0:*>5}|".format("Hi"))
print("{:,} {:.2%}".format(1234, 0.245))
print("{{{}}}".format(123)) # 중괄호 자체를 문자로 인쇄 -> {123}
```

### [1-24] f-string (f-prefix)을 이용한 최신 인쇄 방식

- 문자열 앞에 알파벳 f를 붙이는 방식으로, 파이썬 3.6부터 도입되어 현재 가장 대중적이고 직관적인 포맷팅 방법입니다.
- 변수 이름을 문자열 안에 그대로 넣을 수 있고, 중괄호 안에서 간략한 연산({age - 10})도 가능합니다. 형식 지정 규칙은 .format()과 동일하게 작동합니다.
```python
name = "mono"
age = 20
f1 = 11.3456

s = f"{name}, {age - 10}" # 중괄호 내부에서 산술 연산 가능
print(s)

print(f"{f1:10.5f}")       # 10칸 확보 후 소수점 5자리 실수 출력
print(f"name = {name}")
print(f"{age:>5}-{age:0>5}") # 5칸 우측 정렬 / 5칸 우측 정렬 후 빈칸 0 채움
print(f"{age:<5}-{age:0^5}") # 5칸 좌측 정렬 / 5칸 가운데 정렬 후 빈칸 0 채움
print(f"{123456789:,}")     # 천 단위 콤마
print(f"{0.125:.2%}")       # 백분율 변환 (12.50%)
```
